Esse repositório tem o intuito de salvar as alterações realizadas em processos que não usam o git para atualização, como a API ou personalizações.

Sempre que realizar uma alteração que for efetivamente para produção, realizar um novo commit para manter salvo o que foi alterado e se precisar reverter, terá essa possibilidade.

Mantenha o motivo das mudanças bem explicados para facilitar futura manutenção.
